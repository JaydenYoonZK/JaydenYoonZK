## Hi, I'm Jayden

I build web tools for the practical parts of running an online business: client portals, billing screens, admin panels, content utilities, and the small automations that save a team from repeat work.

My public GitHub is still being cleaned up. I am starting with small projects that have clear docs and a simple test path, especially when they come from something I had to solve myself.

## What I'm working on

- [whmcs-emoji-compatibility-guide](https://github.com/JaydenYoonZK/whmcs-emoji-compatibility-guide): a copy board and compatibility reference for WHMCS templates, product names, emails, and client-area text.
- Small hosting-business utilities that come from real client and founder work.
- Short notes and examples for things I have tested myself.
